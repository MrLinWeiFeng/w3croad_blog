# 一些常用的css

1. 移动端点击时，有时出现黑影

```
div{
    -webkit-tap-highlight-color: rgba(0,0,0,0)
}
```

2. input输出框样式

```
input{
    border: none;
    background:none;
}
input:focus{
    outline: none;
}
```
