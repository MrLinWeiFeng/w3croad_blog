# transition

css中的补间动画是由 `transition` 来实现的。

`transition` 是几个属性的集合：

- `transition-property`
- `transition-duration`
- `transition-delay`
- `transition-timing-function`： 缓动函数
    - ease： 慢快慢
    - linear
    - ease-in
    - ease-out
    - ease-in-out：先加速再减速
    - cubic-bezier(n, n, n, n) ，n 在 0-1 之间。
    
关于cubic-bezier的制作：

- http://easings.net/zh-cn
- http://cubic-bezier.com/
