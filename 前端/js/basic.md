## JSDoc注释规范

```
/**
 * @override 对当前代码文件的描述
 * @copyright
 * @author <name> [<emailAddress>] 代码的作者信息
 * @version 当前代码的版本
 */
/**
 * 创建 Hi 实例.
 * @constructor
 * @param {string} name - 姓名.
 * @param {number} age - 年龄.
 * @returns {{name: *, age: *}}
 * @example
 * new Hi('jey', 12)
 */
function Hi(name, age) {
	return {
		name, age
	}
}
```
